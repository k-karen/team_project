## Setup
`docker-compose up`

## 各コマンドの実行方法
`docker exec -it g2_app bash` でrailsが動いてるコンテナに入れます。  
コンテナ内部で `bin/rails g model Hoge` とかすればｒailsコマンドが実行できます。
